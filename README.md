### Data Assignment – Daily Returns Analysis ###

[Stakeholders want to understand how much gross revenue is returned per day, and how this differs between online and offline sales]

- Context

You receive a daily file containing sales transactions from different sales channels.

Each row represents one transaction.

- The data contains:

A transaction date

A transaction number

A gross amount

A return amount.

Sales channel information

- Task

Using the provided data:

1. Produce a daily aggregation that shows, for each date and each channel:

total gross amount

total return amount

The result should be written to a new CSV file.


2. Display the aggregated result in a simple way, e.g. a HTML page with a table or a very simple PHP or JavaScript page.

important: A basic, unstyled table is completely sufficient. Do not spend time on styling or design.



- Deliverables

Please provide:

The resulting aggregated output 

The code used (JavaScript or PHP) to produce it

A short explanation of your approach and any assumptions made


-Time Guidance

This task is designed to take approximately 45 minutes.
